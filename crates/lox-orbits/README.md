# lox-orbits

Spacecraft orbit modelling tools for the [Lox](https://github.com/lox-space/lox) ecosystem,
including Keplerian elements, orbit propagation, and constellation design.

Part of [Lox – Oxidized Astrodynamics](https://github.com/lox-space/lox).
