# lox-odm

Rust types and (de-)serialization for CCSDS Orbit Data Messages (ODM):
OPM, OEM, OMM, OCM, and the ODM-CI combined-instantiation wrapper.
Part of the [Lox](https://github.com/lox-space/lox) ecosystem.

Status: under construction. See
`docs/superpowers/specs/2026-05-14-lox-odm-rewrite-design.md` for the
design that drives this crate.
