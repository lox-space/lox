# lox-frames

Reference frame transformations for the [Lox](https://github.com/lox-space/lox) ecosystem,
including ICRF, ITRF, TEME, and CIO/equinox-based transformation chains.

Part of [Lox – Oxidized Astrodynamics](https://github.com/lox-space/lox).
