# lox-earth

Tools for Earth-based astrodynamics for the [Lox](https://github.com/lox-space/lox) ecosystem,
including Earth orientation parameters, nutation, and precession models.

Part of [Lox – Oxidized Astrodynamics](https://github.com/lox-space/lox).
